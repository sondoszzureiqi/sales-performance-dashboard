# Sales Performance Dashboard (Tableau)
This project features an interactive Tableau dashboard built to analyze and visualize key sales performance metrics across multiple years. Using built-in filters, users can explore sales, profit, quantity, and monthly trends for any selected year, enabling flexible and dynamic analysis.

## Key Highlights
- KPI cards displaying total **sales**, **profit**, and **quantity**
- Year-over-year **performance comparisons**
- Sales & profit **subcategory analysis**
- Monthly **trend lines** with high/low indicators
- Filters to switch between **different years** for deeper insight
- Clean and responsive layout for **desktop and mobile**
  
 ## Tools Used
  - Tableau Public  
  - Data visualization components:
  - **KPI cards**
  - **Sparklines**
  - **Trend lines**
  - **Comparative bar charts**
  - **Year filters**
